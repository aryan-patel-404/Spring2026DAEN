# Docs Catalog

This folder is organized by category to keep research, design, planning, and operations artifacts easy to maintain.

## Structure

- `docs/specs/`
  - Formal design specifications and implementation contracts.
  - Current key files:
    - `docs/specs/Index_Spec_v1.md`
    - `docs/specs/findings_to_index_implementation_playbook.md`
    - `docs/specs/terrain_feature_pack_v1.md`

- `docs/research/`
  - Research prompt stacks, literature review prompts, and question-tracking logs.
  - Current files:
    - `docs/research/research_stack.md`
    - `docs/research/research_question_bank.md`

- `docs/planning/`
  - Project planning documents, scope notes, and outlines.

- `docs/meetings/`
  - Publishable meeting minutes and session records approved for repo inclusion.

- `docs/admin/`
  - Administrative and internal proposal documents.

## Authoring Note

For final products, use neutral language for source attribution (for example, “research findings” or “model-assisted analysis”) unless specific tool attribution is explicitly required.
